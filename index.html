الدرس 1.4: رسوم الدوال النسبية (Graphs of Rational Functions)

Asymptotes, Holes, and Transformations

📅 التاريخ: 👤 الاسم: 🏫 الشعبة: ⏱️ المدة: 60 دقيقة 📊 المجموع: 100 درجة

🔹 الجزء الأول: الخطوط المقاربة الرأسية والمجال

  1. 1. الخط المقارب الرأسي للدالة $f(x) = \frac{1}{x-1}$ هو:
    ✅ الإجابة: ب) x = 1 (عندما يكون المقام صفراً: x - 1 = 0)
  2. 2. الخطوط المقاربة الرأسية للدالة $f(x) = \frac{x}{x^2-16}$ هي:
    ✅ الإجابة: ج) x² - 16 = 0 → (x-4)(x+4) = 0 → x = 4 أو x = -4
  3. 3. متى يظهر خط مقارب رأسي في الدالة النسبية؟
    ✅ الإجابة: أ) المقام = 0 والبسط ≠ 0
  4. 4. مجال الدالة $f(x) = \frac{1}{x+2}$ هو:
    ✅ الإجابة: ب) المجال يستثني القيم التي تجعل المقام صفراً (x = -2)

الجزء الثاني: الخطوط المقاربة الأفقية

  1. 5. الخط المقارب الأفقي للدالة $f(x) = \frac{3x}{x+2}$ هو:
    ✅ الإجابة: ب) درجة البسط = درجة المقام، لذا y = نسبة المعاملات الرئيسية = 3/1 = 3
  2. 6. الخط المقارب الأفقي للدالة $f(x) = \frac{2}{x^2-1}$ هو:
    ✅ الإجابة: أ) درجة البسط (0) < درجة المقام (2)، لذا y = 0
  3. 7. الخط المقارب الأفقي للدالة $f(x) = \frac{x^2+1}{x}$ هو:
    ✅ الإجابة: ج) درجة البسط (2) > درجة المقام (1)، لذا لا يوجد خط مقارب أفقي
  4. 8. إذا كانت درجة البسط تساوي درجة المقام، فإن الخط المقارب الأفقي هو:
    ✅ الإجابة: أ) y = معامل البسط الرئيسي / معامل المقام الرئيسي

🔹 الجزء الثالث: الثقوب (Holes)

  1. 9. إحداثي x للثقب في الدالة $f(x) = \frac{(x-3)(x+2)}{x-3}$ هو:
    ✅ الإجابة: أ) العامل المشترك (x-3) يسبب ثقباً عند x = 3
  2. 10. ما الذي يسبب ظهور "ثقب" (Hole) في رسم الدالة النسبية؟
    ✅ الإجابة: ب) الثقب هو انقطاع قابل للإزالة بسبب عامل مشترك
  3. 11. الصيغة المبسطة للدالة $f(x) = \frac{x^2-16}{x+4}$ هي:
    ✅ الإجابة: أ) $\frac{(x-4)(x+4)}{x+4} = x - 4$
  4. 12. هل للدالة $f(x) = \frac{1}{x-2}$ ثقب؟
    ✅ الإجابة: ب) لا يوجد عامل مشترك بين البسط (1) والمقام (x-2)، لذا لا يوجد ثقب

🔹 الجزء الرابع: التبسيط والتحليل

  1. 13. بسّط التعبير: $\frac{6x^2+11x+4}{2x^2+x}$
    ✅ الإجابة: ب) البسط $(2x+1)(3x+4)$، المقام $x(2x+1)$، الناتج $\frac{3x+4}{x}$
  2. 14. بسّط التعبير: $\frac{2x^2-8}{x-2}$
    ✅ الإجابة: ب) $\frac{2(x-2)(x+2)}{x-2} = 2(x+2)$
  3. 15. بسّط التعبير: $\frac{x^2-9}{x-3}$
    ✅ الإجابة: ب) $\frac{(x-3)(x+3)}{x-3} = x+3$
  4. 16. أي من الدوال التالية ليس لها ثقب؟
    ✅ الإجابة: ج) لا يوجد عامل مشترك بين البسط (1) والمقام (x)

🔹 الجزء الخامس: الرسوم والتحويلات

  1. 17. الدالة $f(x) = \frac{1}{x-2} + 3$ لها خطوط مقاربة عند:
    ✅ الإجابة: أ) V.A: x-2=0→x=2، H.A: y=3
  2. 18. الدالة الأم (Parent Function) للدوال النسبية الأساسية هي:
    ✅ الإجابة: أ) $f(x) = 1/x$
  3. 19. التحويل في الدالة $f(x) = \frac{1}{x+4} - 1$ مقارنة بالدالة الأم هو:
    ✅ الإجابة: أ) (x+4) يعني يسار 4، و (-1) يعني أسفل 1
  4. 20. نقطة تقاطع الخطوط المقاربة للدالة $f(x) = \frac{1}{x+1} - 2$ هي:
    ✅ الإجابة: ج) V.A: x=-1، H.A: y=-2 → النقطة (-1, -2)
